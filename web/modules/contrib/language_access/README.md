# Language Access

Provide access permissions by language.

## INTRODUCTION

Restrict user access to locale languages using permission.
Useful for sites where multilingual content has not been fully prepared yet.

* For a full description of the module, visit the project page: 
  https://drupal.org/project/language_access
* To submit bug reports and feature suggestions, or to track changes:
  https://drupal.org/project/issues/language_access
  
## REQUIREMENTS

This module requires no modules outside of Drupal core.

## INSTALLATION

Install as you would normally install a contributed Drupal module. Visit:
https://www.drupal.org/docs/8/extending-drupal-8/installing-drupal-8-modules
for further information.

## CONFIGURATION

The module has no menu or modifiable settings. 
There is no configuration except permissions.
